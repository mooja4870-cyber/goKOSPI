# DAY 3 — 시나리오 & 성공 기준

> 출처: [16일간의 기적 · 바이브 코딩 로드맵](https://project.aibizbotu.com/) · 3단계 (DAY 3 / PLAN)

## 요약

사용자 시나리오를 구체화하여 제품의 사용 흐름을 파악하고, 개발 후 검증할 수 있는 정량적인 **성공 기준(정적/동적 성공 기준)**을 정의합니다.

## 목표

- 대표님이 실전 투자에서 겪는 상세 사용자 시나리오를 설계한다.
- 리밸런싱 패턴 매칭 알고리즘의 정성적/정량적 성공 기준을 수립한다.

## 체크리스트

- [x] 사용자 시나리오(Happy Path 및 예외 케이스)를 설계한다.
- [x] 서비스 동작 단계별 상태 전이(State Transition)를 정의한다.
- [x] 비즈니스/기술적 측면의 정량적 성공 기준을 작성한다.
