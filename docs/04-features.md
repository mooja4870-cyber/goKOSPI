# 기능 목록 (Feature List)

`goKOSPI` 리밸런싱 감지 알림 봇의 핵심 모듈 및 기능 목록입니다. 본 시스템은 Go 언어로 구현되며 크게 5가지 모듈로 구성됩니다.

---

## 1. 마켓 데이터 수집기 (Market Data Collector)
* **주기적 시세 동기화**:
  * 코스피 지수(^KS11) 및 시총 상위 50대 종목의 일봉 데이터를 Yahoo Finance 또는 국내 증권사 API를 통해 주기적으로 수집합니다.
* **병렬 수집 엔진 (Goroutines)**:
  * 50대 종목의 과거 20일치 시세 데이터를 속도 유실 없이 다중 스레드로 병렬 요청합니다.
* **수급 데이터 취득**:
  * 당일 외국인, 기관, 개인 투자자별 순매수/순매도 거래대금 데이터를 함께 크롤링/조회합니다.

---

## 2. 변동성 분석 및 시그널 엔진 (Signal Engine)
* **시간 가중 누적 상승률 연산**:
  * 최근 3일간의 시간 가중치($W$)가 반영된 누적 변동폭을 계산합니다.
* **종목별 변동성(ATR) 동적 정규화**:
  * 각 종목의 최근 20일 ATR(Average True Range)을 산출하여 상승폭을 ATR 배율(Z-Score)로 정규화합니다.
* **임계 신호(Trigger) 매칭**:
  * 정규화된 스코어가 종목별 한계값(예: ATR의 2.5배 이상)에 도달했는지 실시간 대조 판별합니다.

---

## 3. 수급 교차 필터 엔진 (Filter Engine)
* **상승 주체 분석**:
  * 과열 신호 포착 시 해당 종목의 외인/기관 순매수 잔고 흐름을 점검합니다.
* **알림 유예(Hold) 판정**:
  * 외인/기관 매수세 강도가 상위 5% 이내일 경우 신호 발생을 지연(Cooldown)시켜 강세장 휩소를 회피합니다.

---

## 4. 디스코드 알림 모듈 (Discord Notifier)
* **웹훅 연동**:
  * 디스코드 API 채널로 마크다운 양식의 임베드 메시지를 전송합니다.
* **알림 등급 세분화**:
  * [과열 주의(Yellow)], [과열 경보/매도 권장(Red)], [조정 완료/매수 권장(Green)] 등 상태를 구분한 컬러 가이드 제공.

---

## 5. 백테스팅 시뮬레이터 (Local Backtester)
* **과거 시뮬레이션 구현**:
  * 과거 3개년 오프라인 CSV/JSON 일봉 데이터를 활용해 동일 조건의 진입/청산 승률 및 수익률을 측정하는 간이 테스트 러너(Runner)를 포함합니다.
