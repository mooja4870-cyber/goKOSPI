## v2.8.0

Date: 2026-07-08

### 변경 내용

* **[혁신 #1] Z-Score 인터랙티브 차트 패널 구현** (TradingView Lightweight Charts 기반)
  * 모든 종목 행 클릭 or 📈 차트 버튼 클릭 → 슬라이드업 차트 모달 팝업
  * 3개월 / 6개월 / 1년 기간 선택 버튼으로 즉시 전환
  * 캔들스틱(상승=녹색/하락=빨간색) + 매수/매도 시그널 마커(화살표) 자동 표시
  * Z-Score 과열도 추이 히스토그램 (빨간=과열, 녹색=저점, 회색=중립) 별도 패널
  * 현재 종가 / Z-Score / 현재 신호 / 마커 건수 통계 요약 카드 표시
* **[혁신 #2] 시장 히트맵 뷰 구현** (Finviz 트리맵 벤치마킹)
  * 스캐너 테이블/히트맵 전환 토글 버튼 (📋 테이블 ↔ 🗺️ 히트맵)
  * 히트맵 셀 = 종목명 + Z-Score 수치 + 신호 레이블, 7단계 신호별 색상 그라디언트
  * 히트맵 셀 클릭 시 차트 모달 즉시 호출 (차트+히트맵 연동)
* **[백엔드] `/api/chart_data` 엔드포인트 신설**
  * OHLCV 캔들 데이터 + Z-Score 시계열 + 시그널 마커를 경량 JSON으로 반환
  * 기존 `engine.py`의 `calculate_rebalancing_signals()` 재활용 (중복 없음)
  * 3mo / 6mo / 1y 기간 파라미터 지원

### 수정 파일

* `templates/index.html` — 차트 모달/히트맵 CSS·HTML·JS 전면 추가
* `app.py` — `/api/chart_data` 라우트 신설

### 비고

* TradingView Lightweight Charts v4.2.0 (CDN, 무료 오픈소스) 사용
* 히트맵은 추가 라이브러리 없이 순수 HTML/CSS/JS로 구현
